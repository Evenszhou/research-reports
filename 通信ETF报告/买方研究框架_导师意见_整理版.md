# 买方研究框架——导师意见整理

---

## 一、买方 vs 卖方的核心差异

| | 卖方 | 买方 |
|---|---|---|
| 核心问题 | 这个东西很好，很棒 | 这个东西很好，很棒，**我怎么挣钱？** |
| 分析终点 | 行业前景看好，建议关注 | **从何门入？挣什么钱？**（分子端/分母端/高景气/拥挤度/政策/AI？） |

## 二、行业分析框架

### 2.1 需求侧分析

**渗透率**
- 当前渗透率是多少？终局渗透率（天花板）是多少？
- 渗透率处于什么阶段：早期导入 → 加速成长 → 成熟 → 衰退？
- 海外渗透率情况：全球（含/不含美日印等非友好市场）分别如何？海外增量来自哪里？渗透率提升速度怎样？
- 折旧视角：产品换机周期多长？当前存量用户中处于换机窗口的比例？

**消费周期**
- 当前消费在峰还是谷？同比、环比趋势如何？
- 消费景气度的高频/中频指标有哪些？

### 2.2 供给侧分析

**量价关系**
- 四种基本形态：量价齐升、量升价跌、量价齐跌、价升量跌
- 周期下行时的特征：即使是产业龙头，价格也会跌到二三线水平
- 关注点：产能利用率、库存水位、新增产能投放节奏

**去化逻辑**
- 下行期有没有去化动力？
  - 有 → 什么在催动去化？
  - 没有 → 什么在阻止去化？（例如：光伏——大家都在熬，等熬死对手赢家通吃）

### 2.3 成长性分析

**增长动力**
- 商家是否有独立提价能力？（如奢侈品行业的品牌溢价）
- 增长是否线性？有没有非线性加速/减速的拐点？

**潜在障碍**
- 政策风险：责任归属问题（智驾事故、机器人伤人、无人机坠毁等）
- 技术风险：技术路径是否确定？
- 供给瓶颈：产能或原材料是否制约增长？

### 2.4 可比产业对标

- 找到处于相同或相似成长阶段的历史产业
- 可比产业在同期二级市场的表现如何？弹性、持续时间、估值水平？

## 三、ETF筛选与分析

### 3.1 初筛：规模与流动性

| 规模档位 | 分类 |
|---|---|
| ≥5亿 | 大盘ETF |
| 2-5亿 | 中盘ETF |
| 5000万-2亿 | 小盘ETF |

- 流动性与规模高度相关，是同一问题的两个面
- 如港股有相似/相同标的，需对比

### 3.2 个票深度分析

**持仓质量**
- 十大持仓情况
- 持仓大票与该主题的偏离度（**一般选偏离小的票**）
- 偏离度判断：持仓是否真正集中在目标赛道

**管理指标**
- 管理人情况
- 跟踪误差
- 管理费率和分红（细枝末节，最后再看）

**历史表现**
- 在过往周期中的弹性如何？
- 周期上行时涨幅？下行时回撤？
- 与行业指数的跟踪关系

## 四、估值与周期定位

### 4.1 周期底部判断

- 过去周期在什么时候见底？因为什么见底？
- 见底的数据来源是什么？数据频率（高频/月频）？
- 行业基本面变化 vs 二级市场表现：谁先见底？谁先出底？
- 行业资本支出能否被观测到？Capex变化是先行还是滞后指标？

### 4.2 量价锚定

- 锚定要看**量、看成长、看周期**
- 从**供给、需求、量、价**四个维度统合分析
- 行业营收趋势、行业指数走势、二级市场起涨状态

## 五、通信行业的特殊分析逻辑

传统行业分析框架适用于"前有先例、后亦承前"的行业。但通信行业因**大模型崛起**发生了根本性变化，需要一套新的自上而下推算逻辑：

### 5.1 分析链条

```
大模型使用人群
  → 不同频率用户的使用行为（推理/代码/文字/视频/计算？）
    → Token消耗量
      → 硬件需求（服务器数量、万卡集群数量、芯片用量）
        → 训练占比 vs 推理占比
          → 通信需求（光模块数量/规格、PCB层数/面积）
```

### 5.2 核心观点

**通信锚定的是新的生态**，而非传统的运营商资本开支周期。量的估计必须从大模型使用人群出发，自上而下推算，而非从行业供给端出发。

## 六、ETF过会报告要点

过会报告需要在行业分析的基础上，回答：
1. **为什么要投这个赛道？**（高景气/政策驱动/技术拐点）
2. **现在是什么位置？**（周期阶段/估值水位/拥挤度）
3. **选哪只ETF？**（规模、偏离度、弹性、流动性）
4. **怎么挣钱？**（分子端/分母端/估值修复/景气扩张）
5. **风险在哪？**（什么情况下会亏钱）
