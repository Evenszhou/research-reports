### 文件职责

#### 1. `__init__.py`
- **作用**：模块入口，定义对外接口
- **导出类**：
  - `Datafeed` → `XtDatafeed`（数据服务）
  - `XtGateway`（交易网关）

```python
from vnpy_xt import Datafeed, XtGateway
```

#### 2. `xt_config.py`
- **作用**：存储全局配置常量
- **关键配置**：
  - `VIP_ADDRESS_LIST`：迅投VIP服务器地址列表
  - `LISTEN_PORT`：本地监听端口（默认58620）

#### 3. `xt_datafeed.py`
- **作用**：实现历史数据查询
- **核心类**：`XtDatafeed`（继承自 `BaseDatafeed`）
- **主要方法**：
  - `init()`：初始化数据服务
  - `query_bar_history()`：查询K线
  - `query_tick_history()`：查询Tick

#### 4. `xt_gateway.py`
- **作用**：实现实时行情和交易
- **核心类**：
  - `XtGateway`：网关主类
  - `XtMdApi`：行情API
  - `XtTdApi`：交易API
- **辅助函数**：
  - `process_etf_option()`：处理ETF期权
  - `process_futures_option()`：处理期货期权
  - `generate_datetime()`：时间戳转换

## 核心类详解

### 一、XtDatafeed（数据服务）

#### 类定义

```python
class XtDatafeed(BaseDatafeed):
    """迅投研数据服务接口"""
    
    lock_filename = "xt_lock"
    lock_filepath = get_file_path(lock_filename)
```

#### 关键属性

- `username`：连接用户名（"token" 或 "client"）
- `password`：Token 或留空
- `inited`：是否已初始化
- `lock`：文件锁（确保单例）

#### 核心方法

##### 1. `init(output=print) -> bool`

**功能**：初始化数据服务连接

**流程**：
```
init()
  ├─ 检查是否已初始化
  ├─ Token模式 → init_xtdc()
  ├─ 客户端模式 → 直接使用
  └─ 测试查询合约信息
```

**源码关键点**：
```python
# Token模式需要启动数据服务进程
if self.username != "client":
    self.init_xtdc()

# 测试连接
xtdata.get_instrument_detail("000001.SZ")  # 查询平安银行
```

##### 2. `query_bar_history(req, output=print) -> list[BarData]`

**功能**：查询K线历史数据

**核心逻辑**：
1. 调用 `get_history_df()` 获取原始数据
2. 遍历数据行，转换为 `BarData` 对象
3. 处理时间调整（结束时刻 → 开始时刻）
4. 合并集合竞价数据（分钟线）

**关键代码片段**：
```python
# 时间调整
dt: datetime = datetime.fromtimestamp(tp.time / 1000)
dt = dt.replace(tzinfo=CHINA_TZ)
dt = dt - adjustment  # 减去时间周期

# 合并集合竞价
if auction_bar and auction_bar.volume:
    bar.open_price = auction_bar.open_price
    bar.volume += auction_bar.volume
```

##### 3. `get_history_df(req, output) -> DataFrame`

**功能**：从迅投获取原始数据

**流程**：
1. 转换参数格式（VeighNa → 迅投）
2. 调用 `xtdata.download_history_data()` 下载数据
3. 调用 `xtdata.get_local_data()` 获取本地数据

**关键映射**：
```python

xt_symbol = symbol + "." + EXCHANGE_VT2XT[exchange]
# 例：000001.SZSE → "000001.SZ"

# 时间周期转换
```

### 二、XtGateway（交易网关）

#### 类定义

```python
class XtGateway(BaseGateway):
```

#### 关键属性

- `md_api`：行情API实例（`XtMdApi`）
- `td_api`：交易API实例（`XtTdApi`）
- `trading`：是否启用交易
- `orders`：委托缓存字典

#### 核心方法

##### 1. `connect(setting)`

**功能**：连接网关（异步）

**流程**：
```python
connect(setting)
  ├─ 创建独立线程
  └─ _connect(setting)
      ├─ 初始化行情API（md_api.connect）
      ├─ 初始化交易API（td_api.connect，可选）
      └─ 启动定时查询任务
```

##### 2. `subscribe(req)`

**功能**：订阅行情

**实现**：
```python
def subscribe(self, req: SubscribeRequest) -> None:
    """订阅行情"""
    # 转换合约代码
    xt_symbol = req.symbol + "." + EXCHANGE_VT2XT[req.exchange]
    
    # 注册回调
    if xt_symbol not in self.subscribed:
        xtdata.subscribe_quote(
            stock_code=xt_symbol,
            period="tick",
            callback=self.onMarketData
        )
```

##### 3. `send_order(req) -> str`

**功能**：委托下单

**实现**（委托给 `td_api`）：
```python
def send_order(self, req: OrderRequest) -> str:
    """委托下单"""
    if self.trading:
        return self.td_api.send_order(req)
    return ""
```

### 三、XtMdApi（行情API）

#### 核心回调：`onMarketData(data)`

**触发时机**：每次收到行情推送

**处理流程**：
```
onMarketData(data)
  ├─ 解析合约代码（symbol + exchange）
  ├─ 获取合约信息（symbol_contract_map）
  ├─ 创建 TickData 对象
  ├─ 填充5档盘口数据
  ├─ 价格精度处理（round_to）
  └─ 推送到网关（gateway.on_tick）
```

**数据结构**：
```python

    "time": 1702345678000,  # 时间戳（毫秒）
    "lastPrice": 10.5,
    "open": 10.3,

    "volume": 1000000,

    "bidPrice": [10.49, 10.48, ...],
    "askPrice": [10.51, 10.52, ...],
    "bidVol": [1000, 2000, ...],
    "askVol": [1500, 2500, ...]
}
```

#### 关键方法：`query_contracts()`

**功能**：查询合约信息并推送

**分类查询**：
1. **股票合约**：`query_stock_contracts()`
   - 市场板块：沪深A股、转债、ETF、指数、京市A股
   - 产品类型：EQUITY、FUND、INDEX

2. **期货合约**：`query_future_contracts()`
   - 市场板块：中金所、上期所、能源中心、大商所、郑商所、广期所
   - 产品类型：FUTURES、OPTION

3. **期权合约**：`query_option_contracts()`
   - ETF期权：调用 `process_etf_option()`
   - 期货期权：调用 `process_futures_option()`

### 四、XtTdApi（交易API）

#### 核心方法

##### 1. `send_order(req) -> str`

**功能**：委托下单

**实现**：
```python
def send_order(self, req: OrderRequest) -> str:
    # 生成本地委托号
    orderid = self.new_orderid()
    
    # 转换参数
    stock_code = req.symbol + "." + EXCHANGE_VT2XT[req.exchange]
    order_type = DIRECTION_VT2XT[(req.direction, req.offset)]
    
    # 发送委托
    self.xt_client.order_stock_async(
        account=self.xt_account,
        stock_code=stock_code,
        order_type=order_type,
        order_volume=int(req.volume),
        price_type=ORDERTYPE_VT2XT[(req.exchange, req.type)],
        price=req.price,
        order_remark=orderid
    )
    
    # 本地缓存委托
    order = req.create_order_data(orderid, self.gateway_name)
    self.gateway.on_order(order)
    
    return order.vt_orderid
```

##### 2. 核心回调

| 回调方法 | 触发时机 | 数据类型 |
|---------|---------|---------|
| `on_stock_trade()` | 成交推送 | `XtTrade` → `TradeData` |
| `on_stock_order()` | 委托回报 | `XtOrder` → `OrderData` |
| `on_query_asset_async()` | 资金查询 | `XtAsset` → `AccountData` |
| `on_query_positions_async()` | 持仓查询 | `XtPosition` → `PositionData` |

### 修改3：增加重连机制

**问题**：连接断开后不会自动重连

#### 修改位置：`xt_gateway.py` 的 `XtTdApi.on_disconnected()`

**原代码**（简单重连）：
```python
def on_disconnected(self) -> None:
    """连接断开"""
    self.gateway.write_log("交易接口连接断开，请检查与客户端的连接状态")
    self.connected = False
    
    # 尝试重连
    session = int(float(datetime.now().strftime("%H%M%S.%f")) * 1000)
    connect_result = self.connect(self.path, self.account_id, self.account_type, session)
    
    if connect_result:
        self.gateway.write_log("交易接口重连失败")
    else:
        self.gateway.write_log("交易接口重连成功")
```

**优化后**（增加重试和延迟）：
```python
def on_disconnected(self) -> None:
    """连接断开"""
    self.gateway.write_log("交易接口连接断开")
    self.connected = False
    
    # 重试3次，每次间隔5秒
    max_retry = 3
    retry_delay = 5
    
    for i in range(max_retry):
        self.gateway.write_log(f"尝试重连 ({i+1}/{max_retry})...")
        time.sleep(retry_delay)
        
        session = int(float(datetime.now().strftime("%H%M%S.%f")) * 1000)
        connect_result = self.connect(self.path, self.account_id, self.account_type, session)
        
        if not connect_result:
            self.gateway.write_log("交易接口重连成功")
            return
    
    self.gateway.write_log("交易接口重连失败，请手动重启")
```

## 实战改造案例

### 案例1：增加数据缓存机制

**目标**：避免重复下载相同的历史数据

#### 实现方案

```python
# 在 xt_datafeed.py 中添加缓存类
from datetime import datetime, timedelta
from typing import Optional
import pickle
import os

class DataCache:
    """数据缓存管理器"""
    
    def __init__(self, cache_dir: str = "./cache"):
        self.cache_dir = cache_dir
        os.makedirs(cache_dir, exist_ok=True)
    
    def get_cache_key(self, req: HistoryRequest) -> str:
        """生成缓存键"""
        return f"{req.symbol}_{req.exchange.value}_{req.interval.value}_{req.start}_{req.end}"
    
    def get(self, req: HistoryRequest) -> Optional[list]:
        """从缓存读取"""
        cache_file = os.path.join(
            self.cache_dir, 
            f"{self.get_cache_key(req)}.pkl"
        )
        
        if not os.path.exists(cache_file):
            return None
        
        # 检查缓存时效（1天过期）
        if time.time() - os.path.getmtime(cache_file) > 86400:
            return None
        
        with open(cache_file, 'rb') as f:
            return pickle.load(f)
    
    def set(self, req: HistoryRequest, data: list):
        """写入缓存"""
        cache_file = os.path.join(
            self.cache_dir,
            f"{self.get_cache_key(req)}.pkl"
        )
        
        with open(cache_file, 'wb') as f:
            pickle.dump(data, f)

# 修改 XtDatafeed 类
class XtDatafeed(BaseDatafeed):
    def __init__(self) -> None:
        super().__init__()
        # ... 原有代码 ...
        self.cache = DataCache()  # 新增缓存
    
    def query_bar_history(self, req: HistoryRequest, output: Callable = print) -> list[BarData]:
        """查询K线数据"""
        # 尝试从缓存读取
        cached = self.cache.get(req)
        if cached:
            output(f"从缓存读取数据: {req.symbol}")
            return cached
        
        # 正常查询
        history = []  # ... 原有逻辑 ...
        
        # 写入缓存
        if history:
            self.cache.set(req, history)
        
        return history
```

### 案例2：增加行情录制功能

**目标**：实时保存Tick数据到文件

#### 实现方案

```python
# 在 xt_gateway.py 中添加录制器
import csv
from datetime import datetime

class TickRecorder:
    """Tick数据录制器"""
    
    def __init__(self, output_dir: str = "./tick_data"):
        self.output_dir = output_dir
        os.makedirs(output_dir, exist_ok=True)
        self.files: dict[str, csv.writer] = {}
    
    def record(self, tick: TickData):
        """记录Tick"""
        # 按合约和日期分文件
        date_str = tick.datetime.strftime("%Y%m%d")
        filename = f"{tick.symbol}_{date_str}.csv"
        filepath = os.path.join(self.output_dir, filename)
        
        # 检查文件是否已打开
        if filename not in self.files:
            file = open(filepath, 'a', newline='')
            writer = csv.writer(file)
            
            # 写入表头
            if file.tell() == 0:
                writer.writerow([
                    'datetime', 'last_price', 'volume', 'turnover',
                    'bid1', 'bid2', 'bid3', 'bid4', 'bid5',
```

```python
# 在 xt_gateway.py 中添加风控类
class RiskManager:
    """风险管理器"""
    
    def __init__(self):
        self.max_order_volume = 10000      # 单笔最大委托量
        self.max_order_value = 100000      # 单笔最大委托金额
        self.max_daily_orders = 100        # 每日最大委托次数
        self.daily_order_count = 0         # 当日委托计数
        self.order_count_date = None       # 计数日期
    
    def check_order(self, req: OrderRequest, gateway) -> bool:
        """检查委托是否合规"""
        # 检查委托量
        if req.volume > self.max_order_volume:
            gateway.write_log(f"委托量超限: {req.volume} > {self.max_order_volume}")
            return False
        
        # 检查委托金额
        order_value = req.price * req.volume
        if order_value > self.max_order_value:
            gateway.write_log(f"委托金额超限: {order_value:.2f} > {self.max_order_value}")
            return False
        
        # 检查当日委托次数
        today = datetime.now().date()
        if self.order_count_date != today:
            self.daily_order_count = 0
            self.order_count_date = today
        
        if self.daily_order_count >= self.max_daily_orders:
            gateway.write_log(f"当日委托次数超限: {self.daily_order_count} >= {self.max_daily_orders}")
            return False
        
        # 通过检查
        self.daily_order_count += 1
        return True
```
